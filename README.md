# Specta

A JupyterLab extension for providing a rich editing experience for [Specta](https://www.mdanalysis.org/specta/) files.

## Features

- Syntax highlighting for Specta files.
- Interactive plotting of spectra.
- Integration with JupyterLab's file browser and editor.

## Installation

```bash
pip install specta
jupyter labextension install specta
```

## Usage

1. Install the extension following the instructions above.
2. Open a Specta file (.spc) in JupyterLab.
3. Enjoy the enhanced editing experience!
